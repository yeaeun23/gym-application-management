# gym-application-management
헬스장 등록 관리 시스템

## 1. 사용자 페이지

### 1-1. 기능
  - 다음 달 신규 등록 or 자동 연장 or 해지하기 (부대시설 선택)
  - 지난 달 등록 내역 보기
  - 헬스장 정보 보기

## 2. 관리자 페이지

### 2-1. 기능
  - 등록 마감하기
  - 등록자 명단 보기 (등록/이름/사번/부서순 조회)
  - 등록자 강제 해지하기
  - 등록자 엑셀 파일로 추출하기
  - 헬스장 정보 입력하기
  - 부대시설 입력하기
